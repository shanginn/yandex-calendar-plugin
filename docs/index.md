# Yandex Calendar для Codex

Неофициальный open-source плагин, который позволяет Codex читать и изменять события Яндекс Календаря через CalDAV.

- [Инструкция и установка](https://github.com/shanginn/yandex-calendar-plugin#readme)
- [Состав первого MVP](mvp.html)
- [Политика конфиденциальности](privacy.html)
- [Условия использования](terms.html)
- [Поддержка](support.html)
- [Подготовка к универсальному каталогу](marketplace-submission.html)

Локальная версия соединяется с Яндексом напрямую и не использует сервер разработчика.
