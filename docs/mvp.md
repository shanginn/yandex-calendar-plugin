# Объём production-ready MVP 0.1

## Цель

Дать одному пользователю Codex безопасный локальный доступ к своему Яндекс Календарю без передачи пароля приложения серверу разработчика.

## Включено

| Сценарий           | MCP-инструмент    | Критерий готовности                                      |
| ------------------ | ----------------- | -------------------------------------------------------- |
| Список календарей  | `list_calendars`  | Возвращает имя, URL и цвет, если он есть                 |
| События за период  | `list_events`     | Диапазон ISO 8601 до 366 дней, структурированный ответ   |
| Создание события   | `create_event`    | Заголовок, начало, конец, описание, место, Телемост      |
| Изменение события  | `update_event`    | Частичное изменение, ETag-защита от потери обновлений    |
| Удаление события   | `delete_event`    | Только с `confirm=true`, destructive-аннотация           |
| Удаление календаря | `delete_calendar` | Свежий точный URL, confirm, защита default/last calendar |

## Нефункциональные требования

- Node.js 20+; одна автономная сборка без установки зависимостей у пользователя.
- HTTPS по умолчанию и запрет произвольного origin из аргументов инструмента.
- Пароль приложения только из окружения или приватного файла `0600`, без логирования в MCP.
- Понятные русские ошибки без ответа CalDAV, содержащего пользовательские данные.
- Строгая схема входов и структурированных выходов.
- Unit-тесты iCalendar и конфигурации, интеграционный тест всего MCP/CalDAV-цикла.
- Валидные манифесты Codex plugin и marketplace.
- CI на pull request и push.

## Не включено

- создание и изменение повторяющихся серий;
- участники, приглашения, ответы и поиск переговорных;
- автоматический поиск свободных интервалов между несколькими людьми;
- несколько Яндекс-аккаунтов в одном локальном процессе;
- облачное хранение credentials;
- публичный OAuth-сервис и хостинг remote MCP.

Эти ограничения выбраны намеренно: первые четыре требуют расширенной совместимости CalDAV, а последние два меняют модель безопасности и необходимы только для универсального каталога ChatGPT/Codex.
